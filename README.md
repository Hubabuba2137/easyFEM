## easyFEM

EasyFEM lets you solve nonstationary heat reansfer problem using finite element method to showcase its principles.

## How to build 

You need a c/c++ compiler, Raylib and CMAKE.

    mkdir build

    cd build

    cmake ..

    cmake --build .

    ./Debug/easyFEM.exe
